# NexaAI 🤖

NexaAI is an AI-powered chat application that enables users to interact with an intelligent AI assistant, securely authenticate their accounts, and access their chat history anytime. The application provides personalized conversations by storing chat records for each authenticated user.

## ✨ Features

* AI-powered conversations using Google Gemini API
* Secure User Authentication & Authorization
* Cookie-Based Authentication
* Protected Routes using Middleware
* Chat History Storage & Retrieval
* Personalized User Conversations
* Responsive and Modern User Interface

## 🛠️ Technologies Used

### Frontend

* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### AI Integration

* Google Gemini SDK API

### Authentication & Security

* Cookies for Secure Authentication
* Middleware for Route Protection

## ⚙️ How It Works

1. Users sign up or log in to their account.
2. Authentication is handled securely using cookies.
3. Protected routes ensure only authenticated users can access their data.
4. User messages are sent to the Google Gemini API.
5. AI-generated responses are returned instantly.
6. Conversations are stored in MongoDB and can be accessed later through chat history.

## 🔒 Security Features

* Secure Cookie-Based Authentication
* Protected API Routes
* User-Specific Data Access Control
* Environment Variable Protection
